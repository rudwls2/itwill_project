# MANGO (음악 스트리밍 사이트)
음악 스트리밍 사이트, Spring 프로젝트
<br/><br/><br/>

## 프로젝트 소개
* 현재 운영되고 있는 멜론, VIVE 등 음악 스트리밍 사이트를 모방한 사이트입니다. 
* 음원, 콘서트 등을 확인하고, 구매하기, 음반 투표 등 실제 페이지의 기능들을 모두 구현했습니다.
